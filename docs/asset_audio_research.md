# Amaris — 2D Isometric Asset & Audio Research

## Context
Amaris is a dark fantasy isometric dungeon crawler (Godot 4.6). Phase 2 (player/movement) is complete. The biggest visual upgrade now is replacing placeholder shapes with real sprite assets, and adding sound/music.

**Git repo:** `https://github.com/ragnorak02/dungeonCrawler.git`
